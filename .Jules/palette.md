# 🎨 Palette's Journal - UX & Accessibility Learnings

## 2024-05-18 - Functional Upload Placeholders
**Learning:** Static "Drag and drop" boxes in decentralized apps often lack the actual JS implementation, leading to user frustration. Transforming these into functional components with clear visual feedback (hover/drag states) and accessibility (ARIA/keyboard) significantly improves perceived reliability and trust in the platform.
**Action:** Always implement full interaction cycles for upload zones, including keyboard triggers, visual feedback during drag events, and clear confirmation of selected files.
