# 🎨 Palette's UX Journal

## 2024-05-15 - Initializing Palette on RestoChain
**Learning:** The project uses static HTML with Tailwind CSS. Forms are missing basic accessibility attributes like `id` and `for` associations, and interactive elements like the dark mode toggle are currently non-functional placeholders.
**Action:** Implement foundational accessibility and add micro-interactions for blockchain-related actions (e.g., submission feedback) to bridge the gap between Web3 complexity and Web2 usability.
