## 2024-05-22 - Documentation as Interface
**Learning:** In repositories where the primary application code (HTML/TSX) has been removed, the documentation (README.md) becomes the primary interface. UX improvements should then focus on scannability, navigation, and visual hierarchy within the documentation.
**Action:** When source code is absent, use Table of Contents, descriptive emojis, and clear sectioning to improve the document UX. Ensure Markdown anchors account for GitHub's stripping of emojis in header IDs.
