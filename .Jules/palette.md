# Palette's UX & Accessibility Journal

## 2025-05-15 - Improving Submission Interactivity
**Learning:** For custom interactive elements like upload zones, combining ARIA roles, tabindex, and keyboard handlers is essential for accessibility. IIFEs help prevent global scope pollution in static HTML projects.
**Action:** Always implement role="button", tabindex="0", and Enter/Space key handlers for custom interactive divs.
