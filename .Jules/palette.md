# Palette's Journal - Critical UX & Accessibility Learnings

## 2024-05-15 - Interactive Submission Feedback
**Learning:** For blockchain-related actions, providing multi-stage feedback (Signing -> Confirming -> Success) significantly reduces user anxiety about "stuck" transactions and prevents double-submissions.
**Action:** Use simulated multi-stage loading states for any action that involves a simulated "on-chain" transaction.
