## 2025-05-14 - [Interactive File Upload Accessibility]
**Learning:** When transforming a static placeholder into an interactive file upload zone, ensuring keyboard accessibility (tabindex, role="button", Enter/Space key listeners) and clear visual feedback for drag-and-drop actions is crucial for a complete UX improvement.
**Action:** Always include keydown handlers and visual state changes (like Tailwind border/bg colors) when creating custom interactive containers.
