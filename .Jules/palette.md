## 2024-05-22 - [Multi-state Button Transitions and Theme Toggle]
**Learning:** In static HTML projects, simple JavaScript (wrapped in an IIFE) can significantly enhance the perceived performance of asynchronous actions (like blockchain transactions) through multi-state button transitions. Using `aria-busy` and `localStorage` for theme persistence provides a robust and accessible foundation for micro-UX.
**Action:** Always implement immediate visual feedback for primary actions and ensure dark mode states are persistent across sessions even in simple prototypes.
