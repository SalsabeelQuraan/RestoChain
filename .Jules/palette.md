## 2025-05-14 - Accessible Theme Toggling and Form Pairing
**Learning:** In static HTML projects, interactive elements like theme toggles often lack initial functional logic and ARIA attributes. Explicitly pairing labels with inputs via 'for' and 'id' is a high-impact, low-effort accessibility win.
**Action:** Prioritize functional dark mode toggles with proper icons (sun/moon) and ensure all form fields have associated labels using the 'for' attribute.
