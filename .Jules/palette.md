# Palette's Journal - Critical Learnings Only
## 2025-05-15 - README Navigation and Scannability
**Learning:** In documentation-heavy repositories where the README is the primary interface, adding a Table of Contents and emojis improves scannability. However, navigation links like '[↑ Back to Top]' should be restricted to major section breaks (e.g., before Level-2 headers) to prevent visual clutter and maintain a high signal-to-noise ratio.
**Action:** Limit document navigation links to major section breaks and use emojis consistently in both TOC and headers to maintain visual harmony.
