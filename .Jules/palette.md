
## 2024-05-20 - [Accessibility & Component Functionality]
**Learning:** In static HTML templates, combining form accessibility (labels, IDs, required attributes) with functional enhancements (drag-and-drop file upload) significantly improves the perceived quality and usability of the interface. However, implementing both in a single pass can push the change count close to the 50-line limit.
**Action:** Prioritize core accessibility (ARIA labels, input associations) first, and ensure functional scripts are as concise as possible using array destructuring and IIFEs to stay within review constraints.
