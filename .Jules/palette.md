## 2026-04-12 - [Accessible Custom File Upload]
**Learning:** Using a hidden file input linked to a focusable label with ARIA roles and keyboard listeners provides a much better UX than the default browser input or static placeholders. Real-time filename feedback and drag-and-drop visuals make the interaction feel 'complete' and trustworthy, especially in a Web3 context.
**Action:** Always implement this pattern for file uploads: focusable label as container, sr-only input, status area for feedback, and drag-and-drop visual states.
